# Arithmagic

### A Roguelike Wizardry Puzzle Game

---

## About
Arithmagic challenges players to combine runes and cast spells in a world of arcane secrets.

In a unique blend of spell-weaving puzzles and deep roguelike strategy, players must fight their way through procedurally generated challenges.
